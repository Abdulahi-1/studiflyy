How I used AI on this project (Studifly)

Putting icons in

I wanted little icons next to the features and in the header but I didn't
really know how to do icons. AI told me I could just use emojis directly
(like 📝 🧠 📅 🦋) instead of installing some icon library, which was way
easier. The feature emojis are stored in app.py (the FEATURES list) and
shown on the home page. The arrow on the Get Started button is just a →
in a little span so I could size it.


Making the emojis move

The background emojis (clouds, butterflies, stars) looked boring just sitting
there so I asked AI how to make them float around. It showed me CSS
@keyframes and the animation property. That's where the clouds drifting,
butterflies floating, and stars twinkling come from (all in styles.css). It
also showed me how to use animation-delay and change the speed for each one
so they don't all move at the same time, which looked weird at first.


Getting the AI answers into a box

At first the summary and quiz text just dumped onto the page and looked bad.
I asked AI how to make it show up in a nice box. It helped me make the
.ai-result class in styles.css (white box, rounded corners, shadow, padding).
The main thing I learned was white-space: pre-wrap so the line breaks from
the AI actually show up instead of everything getting smushed into one
paragraph. The text comes from ai_helper.py and gets put in the box on the
summaries and quizzes pages.


Other stuff

I also asked AI random questions while building, like how Flask routes work,
how the base.html template thing works, and some CSS layout help. I didn't
just copy everything, I changed it to fit my project and made sure I
understood it.
