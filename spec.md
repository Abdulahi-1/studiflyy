
This project focuses on creating an AI Study Buddy that helps college students manage coursework, organize study time, and review course material more effectively. Many students currently use tools such as Canvas LMS and digital calendars to track assignments and deadlines, but these platforms mainly store information rather than actively helping students learn or study. The goal of this project is to combine scheduling and AI-powered learning support into one system that makes studying more manageable and less overwhelming. The application will allow users to upload study materials such as lecture notes or PDFs, enter assignment deadlines, and connect their schedules using the Google Calendar API. The system will then use Hugging Face APIs and transformer models to summarize notes, identify important concepts, and generate practice quiz questions for active learning. Specifically, the project will use the Hugging Face Inference API with models such as facebook/bart-large-cnn for text summarization and google/flan-t5-base for quiz question generation. Uploaded notes will first be converted into plain text, then sent to the Hugging Face API for processing. The generated summaries and quiz questions will then be displayed in the application dashboard. The scheduling component of the project will prioritize assignments based on due dates and estimated workload. Using this information, the system will generate suggested study sessions and automatically add them to the user’s Google Calendar through the Google Calendar API. The scheduling logic will be rule-based rather than AI-generated in order to keep the project realistic and achievable within the remaining time in the quarter. For example, assignments with closer deadlines or higher estimated workloads will receive longer or more frequent study sessions. The application will be implemented using Flask for the backend and frontend routing. Flask will handle file uploads, communication with the Hugging Face API, user input processing, and Google Calendar integration. HTML, CSS, and JavaScript will be used to create a simple dashboard interface where users can view assignments, generated summaries, practice quiz questions, and scheduled study sessions. The project will be completed in several stages throughout the quarter. During Week 1, the basic Flask application structure and user interface will be created, including assignment input and note upload functionality. During Week 2, the Hugging Face summarization and quiz generation APIs will be integrated and tested with uploaded study materials. During Week 3, Google Calendar integration and automated study session scheduling will be implemented. During Week 4, the full system will be tested, debugged, and improved based on usability and functionality. Additional time will also be used to improve the dashboard interface and finalize documentation. Overall, this project is intended to support students who struggle with balancing multiple classes by helping them stay organized while also improving how they study and retain information. By combining scheduling tools with AI-generated study support, the system aims to create a more interactive and effective learning experience for students.
