This project focuses on creating an AI Study Buddy that helps college students manage their coursework, organize their time, and study more effectively. Many students use platforms like Canvas LMS and digital calendars to keep track of assignments, but these tools mainly store information rather than helping students understand or review course material. The goal of this project is to combine scheduling and learning support into one system that makes studying more manageable and less overwhelming. Students will be able to upload notes or study materials, enter assignment deadlines, and connect their schedules through the Google Calendar API. Using AI tools from Hugging Face, the system will summarize notes, identify important concepts, and generate practice quiz questions to encourage active learning. The system will also prioritize assignments based on deadlines and workload, then automatically create personalized study sessions that fit into the user’s calendar. The final output will include summarized notes, generated quizzes, and a calendar-based study plan displayed in a simple dashboard interface. This format is useful because it gives students both an organized schedule and interactive study tools in one place.
The project will be completed in several stages throughout the quarter. First, the basic scheduling system and user interface will be created so users can input assignments and deadlines. Next, the AI note summarization and quiz generation features will be added using the Hugging Face API. After that, Google Calendar integration will be implemented so study sessions can automatically appear on a student’s calendar. Finally, the system will be tested and improved based on usability and functionality. Overall, this project is intended to support students who struggle with balancing multiple classes by helping them stay organized while also improving how they study and retain information.
